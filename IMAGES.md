# CNMO Website — Image Asset Plan

> **How to use this file**
> 1. Open each download URL in your browser — the image will either display or prompt a download.
> 2. Save each file into the `images/` folder using the exact filename listed (makes code wiring easy later).
> 3. Once all images are uploaded, let Claude know and the code will be updated automatically.
>
> **Licenses** — Unsplash License and Pexels License both allow free commercial use. Attribution is not legally required but is good practice (photographer names are listed).
>
> **Executive headshots** — Stock photos cannot be used for real named people. Those 14 slots are listed at the bottom and require actual photos supplied by the executives themselves.
